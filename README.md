Estava querendo fazer alguma coisa diferente em Advpl, com isso pensei em fazer esse jogo, para fugir um pouco de ficar desenhando tela, relatório e integrações.

# Para jogar

Para jogar é utilizado o teclado:

W - Para cima <br/>
A - Para Direita <br/>
S - Para baixo <br/>
D - Para esquerda <br/>
Espaço - Dispara <br/>


# Sobre as imagens
